## Dev Notes

### Metrics Calculation [Source: architecture.md#5.1]

```
GET /api/metrics → {
  total: number,          // count all prospects
  byStatus: {             // count per status
    new: number,
    message_generated: number,
    sent: number,
    replied: number,
    converted: number,
    discarded: number
  },
  responseRate: number,   // (replied + converted) / sent * 100
  conversionRate: number, // converted / total * 100
  messagesSent: number    // count interactions where type = 'message_sent'
}
```

### CSV Export [Source: architecture.md#5.1]

```
GET /api/export/csv?status=X&search=Y → CSV file
Content-Disposition: attachment; filename="prospects-export.csv"
Content-Type: text/csv
```

CSV columns: Nome, Cargo, Empresa, Setor, LinkedIn URL, Status, Última Mensagem, Data de Captura
