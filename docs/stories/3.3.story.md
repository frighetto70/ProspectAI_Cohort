## Story

> I want a complete view of each prospect with full interaction history,
> so that I can make informed decisions about next steps.

### Interaction Timeline

- Ordenar por `created_at` descendente (mais recente primeiro)
- Cada item: ícone por tipo + badge (message_generated, message_sent, reply, note) + conteúdo + data relativa
- Mensagens geradas: mostrar conteúdo completo + metadata (modelo, tokens)
- Notas: mostrar conteúdo com estilo diferente (background sutil)

### Prospect Navigation

- Buscar IDs do prospect anterior e próximo (ordered by created_at ou id)
- Botões "← Anterior" e "Próximo →" no topo da página
- Se não houver anterior/próximo, desabilitar botão

### Prospect Profile Display

Campos a mostrar:
- Nome (grande, h1)
- Cargo + Empresa (h2)
- Setor, Localização
- Headline (itálico)
- Resumo (parágrafo)
- LinkedIn URL (link externo, ícone)
- Status (badge colorido)
- Data de captura

### Quick Actions Bar

Barra horizontal com botões:
- "Gerar Mensagem" → scroll para MessageGenerator
- "Alterar Status" → dropdown com todos os status
- "Adicionar Nota" → abre formulário inline
- "Descartar" → confirmar + Server Action discardProspect
