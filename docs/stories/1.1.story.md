## Dev Notes

### Tech Stack [Source: architecture.md#2]

- **Framework:** Next.js 15 (App Router), TypeScript 5.7, strict mode
- **Styling:** Tailwind CSS 4 + shadcn/ui (copiar via `npx shadcn@latest add`)
- **Testing:** Vitest 3
- **Linting:** ESLint 9 + eslint-config-next

### Project Structure [Source: architecture.md#3]

```
app/
├── layout.tsx          # Root layout (nav, providers)
├── page.tsx            # Placeholder "Dashboard" page
├── loading.tsx         # Loading skeleton
├── login/
│   └── page.tsx        # Login form (password only)
└── api/
    └── health/
        └── route.ts    # GET /api/health → { status: "ok" }

components/
├── ui/                 # shadcn/ui (Button, Card, Input)
└── nav.tsx             # Sidebar navigation

lib/
└── config.ts           # Env vars validation (Zod)

middleware.ts            # Auth middleware (password cookie check)
```

### Authentication [Source: architecture.md#7]

```typescript
// middleware.ts
// - Skip: /api/health, /login, /_next/static, /_next/image, /favicon.ico
// - Check: cookie 'auth' === process.env.APP_PASSWORD
// - Redirect: → /login if not authenticated
```

### Environment Variables [Source: architecture.md#9]

```bash
TURSO_DATABASE_URL=     # Turso database URL
TURSO_AUTH_TOKEN=        # Turso auth token
APIFY_TOKEN=             # Apify API token
GEMINI_API_KEY=          # Google Gemini API key
GEMINI_MODEL=gemini-2.0-flash
APP_PASSWORD=            # Dashboard access password
```
