# Story 2.1: Gemini API Integration & Message Generation

## Status: Ready for Review

## Story

> As a consultant,
> I want personalized prospecting messages generated by AI,
> so that each outreach feels crafted and relevant to the prospect's context.

## Epic

Epic 2: AI Message Generation

## Acceptance Criteria

- [ ] 1. Integração com Gemini API via `@google/generative-ai` SDK usando `GEMINI_API_KEY`
- [ ] 2. Route Handler `POST /api/prospects/[id]/generate-message` gera mensagem personalizada
- [ ] 3. Prompt inclui: dados do prospect (nome, cargo, empresa, headline), contexto do serviço "Strategic Reset", tom desejado (consultor sênior, não vendedor)
- [ ] 4. Resposta inclui mensagem gerada e metadata (modelo usado, tokens consumidos)
- [ ] 5. Mensagem salva na tabela `interactions` com type `message_generated`
- [ ] 6. Possibilidade de regenerar com variações (novo request ao mesmo endpoint)
- [ ] 7. Throttling para respeitar rate limits do Gemini (1s entre chamadas)
- [ ] 8. Tempo de geração < 10 segundos por mensagem
- [ ] 9. Testes para construção do prompt e parsing da resposta

## Dev Notes

### Gemini Client [Source: architecture.md#6.2]

```typescript
// lib/gemini.ts
import { GoogleGenerativeAI } from '@google/generative-ai';
const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY!);
// Model: gemini-2.0-flash (configurable via GEMINI_MODEL env var)
```

### Default System Prompt [Source: architecture.md#6.2]

```
Você é um consultor estratégico sênior especializado em transformação
corporativa. Escreva mensagens de prospecção no LinkedIn que sejam:
- Pessoais e específicas ao contexto do prospect
- Tom de colega estratégico, NUNCA de vendedor
- Curtas (máx 300 caracteres para connection request)
- Mencione algo específico do perfil do prospect
- Apresente o conceito 'Strategic Reset' de forma natural
- Sem emojis, sem exclamações excessivas, sem clichês de vendas
```

### User Prompt Template [Source: architecture.md#6.2]

```
Dados do prospect:
- Nome: ${prospect.name}
- Cargo: ${prospect.title}
- Empresa: ${prospect.company}
- Setor: ${prospect.industry}
- Headline: ${prospect.headline}
- Resumo: ${prospect.summary}

Tipo de mensagem: ${template.type}
${customContext ? `Contexto adicional: ${customContext}` : ''}

Gere a mensagem de prospecção.
```

### Response Format [Source: architecture.md#5.2]

```json
{
  "id": 42,
  "prospectId": 7,
  "type": "message_generated",
  "content": "Olá Ricardo, vi que você está liderando...",
  "metadata": {
    "model": "gemini-2.0-flash",
    "tokensUsed": 287,
    "generationTimeMs": 3200,
    "templateUsed": "connection_request"
  },
  "createdAt": "2026-03-06T14:30:00Z"
}
```

### Throttling

Rate limiter simples: delay mínimo de 1 segundo entre chamadas ao Gemini. Usar variável de controle no módulo (lastCallTimestamp).
